OVERVIEW
The Notification feature is a type of messaging system. The system will send and receive notifications, but is not limited to text. It is also possible to send and receive objects and collection.
Steps to Send a Notification
1) Register the channel by entering the channel name in the "Channel Name" textbox. The channel name can be any name but must be unique and only a single word is allowed
2) Click the "Register" button
3) Enter text in the "Transmit Message(s)" textbox
4) Click the send button
Steps to Receive a Notification
1) Register the channel, as you did in step 1 above
2) From the "Receiving Subscription(s)" list box, select which channel(s) that you want to receive notifications from. This can be your own channel
3) Now when the channels that you have subscribed to send notifications, you will see the channel name along with the message, in the "Received Message(s)" textbox

Composing a Text Message
You can enter text without the keyboard by using the "Speech to Text" converter. To the right of the "Transmit Message(s)" textbox is a microphone icon. Click this icon and begin speaking. Your speech will be converted to text.

Voice Activated Commands
You can activate commands without the keyboard or mouse by using the "Speech to Text" converter. Click the microphone and begin speaking. Your voice commands will initiate the notification commands.

Here is an example registering a channel named "Jupiter".
1) Click on the microphone icon
2) Speak: "Computer, register channel Jupiter"

Other Voice Activate Commands
"Computer, subscribe to channel Jupiter"
"Computer, unregister channel Jupiter"
"Computer, send message"
"Computer, clear text"
