This feature is very unique because its purpose is to enhance the SDLC. These Development Utilities will simplify the Software Developer’s workflow.

Note: This feature is only visible in a Development environment. It is not visible when you are using a Visual Studio Release Configuration.

The Development Utilities:
    • Choose the Angular startup and build settings
    • View the state management
    • View the server-side events, such as unhandled exceptions
    • view performance metrics
